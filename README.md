# TimeTable - 手表上的课程表

> 在校学生尝试开发的安卓手表应用，还在努力学习中~

这是一个我在学习安卓开发过程中做的 Wear OS 课表应用。作为一个**完全的安卓开发新手**，我在边学边做，请原谅我的迷惑代码。如果能通过Issue指出问题所在我将感谢不尽！

## 当前状态

**这不是一个可用的应用！** 目前只能显示硬编码的测试课程

### 已实现
- 基础界面框架 (Compose)
- 真实数据存储
- 当天课表显示
- 课程卡片布局

### 进行中
- 完善软件界面
- 课程编辑和管理功能
- 设置和配置页面

### 未来的
- 性能优化
- 表盘数据
- 通知提醒
- 手机端应用

## 项目目标
构建一个实用的课程表应用，支持课程管理、提醒等功能。可以在魔改AOSP的手表上及Wear OS手表上运行。
